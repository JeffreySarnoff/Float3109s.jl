## A Categorical Algebra for IEEE P3109 Floating-Point Formats

You are a mathematician and computer arithmetic architect working at the intersection of **Applied Category Theory and numerical computing**. Develop a complete categorical framework for the IEEE P3109 floating-point format family — simultaneously a rigorous mathematical specification and a blueprint for a reference implementation. Every implementation artifact should arise as the image of a categorical construction; every abstraction should be witnessed by executable code.

---

### I. Categorical Foundations

Cast the P3109 family as a category **FP** whose **objects** are format tuples $(K, P, \sigma, \mathcal{D})$ (bit-width, precision, signedness, value domain) and whose **morphisms** are structure-preserving maps between code-point spaces respecting the binade lattice, ordering, and special-value conventions.

Define:
- The **code-point functor** $\mathcal{C} : \mathbf{FP} \to \mathbf{Set}$ mapping each format to $\{0,1\}^K$
- The **decoding functor** $\mathcal{V} : \mathbf{FP} \to \mathbf{POrd}$ mapping each format to its partially ordered value set
- Value decoding $\delta : \mathcal{C} \Rightarrow \mathcal{V}$ as a **natural transformation**

### II. Algebraic Structure of Code-Point Space

For each format object:

1. **Partition** the code-point space as a coproduct: $\mathcal{C} \cong \mathcal{C}_{\text{zero}} \sqcup \mathcal{C}_{\text{sub}} \sqcup \mathcal{C}_{\text{norm}} \sqcup \mathcal{C}_{\text{inf}} \sqcup \mathcal{C}_{\text{NaN}}$
2. Define the **binade lattice** $\mathcal{E}(K,P)$ as a poset category and show the exponent field defines a functor $\mathcal{E} : \mathbf{FP} \to \mathbf{Lat}$
3. Establish the **ordinal indexing isomorphism** $\phi : \mathcal{C} \xrightarrow{\sim} \mathcal{O}$ making total order on code points functorial
4. Give **exact rational representations** via a functor $\mathcal{Q} : \mathcal{C}_{\text{fin}} \to \mathbf{Q}$ and prove it factors faithfully through $\mathcal{V}$

### III. Operations, Closure, and Rounding

- Define arithmetic operations as **endomorphisms in FP** (or format-changing morphisms); characterize the resulting operad/PROP structure
- Formalize **closure**: identify the largest subcategory on which each operation is a closed endofunctor; show how NaN/infinity injections restore closure when it fails
- Treat **rounding modes** as a family of natural transformations $\rho_\bullet : \mathcal{V}_\mathbb{R} \Rightarrow \mathcal{V}$ forming a poset ordered by dominance

### IV. Adjunctions and Limits

- Show $\mathcal{V}$ has a **left adjoint** $\mathcal{R} : \mathbf{POrd}_{\overline{\mathbb{R}}} \to \mathbf{FP}$ (the rounding/encoding functor); characterize the unit and counit as round-trip identities
- Show binade structure gives rise to **pullbacks and pushouts** corresponding to format promotion/demotion
- Characterize **initial and terminal objects** of **FP**, or explain what universal constructions substitute for them
